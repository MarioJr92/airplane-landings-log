# Airplane Landings Log

This program is a simple web application built using HTML, CSS, and JavaScript that allows users to log the number of times an airplane lands.

## How to use the program

1. Open the `index.html` file in a web browser.
2. Click the "LAND" button to increment the landing count.
3. Click the "LOG" button to log the previous landings and reset the count to zero.

## Program description

The program consists of an HTML file (`index.html`), a CSS file (`index.css`), and a JavaScript file (`index.js`).

## HTML

The HTML file defines the structure and content of the web page. It contains a heading, two buttons, a counter, and a log of previous landings.

## CSS

The CSS file defines the styles for the HTML elements. It sets the background image, font styles, and button styles.

## JavaScript

The JavaScript file contains the logic for the program. It defines two functions, `increment()` and `save()`, that are called when the "LAND" and "LOG" buttons are clicked, respectively. The `increment()` function increments the landing count, and the`save()` function logs the count and resets it to zero.
